## About This Project
This repository is my submission for the CSE0001 Digital Literacy course project at VIT Bhopal University. As part of the assignment, I took on the role of a Student Digital Ambassador and completed five tasks covering the core areas of digital literacy- awareness, professional presence, coding platforms, communication, and cybersafety.
All five tasks are documented in the project report and supported by screenshots, files, and checklists organised in their respective folders.

## Module Summaries

Task 1 - Digital Literacy Awareness Infographic
Created a one-page infographic using Canva covering what digital literacy is, safe internet practices, and professional online presence.
Task 2 - Student Digital Portfolio
Set up professional profiles on GitHub, LinkedIn, and HackerRank. Created a GitHub profile README with my name, branch, year, and learning goals.
Task 3 - Coding and Collaboration Platforms
Completed a beginner coding challenge on HackerRank and built a five-question Digital Literacy Awareness Quiz using Google Forms.
Task 4 - Professional Email and Etiquette Guide
Drafted two professional emails - one requesting an assignment extension from a professor, and one expressing interest in a summer internship. Also created a social media Do's and Don'ts checklist.
Task 5 - Cybercrime Awareness Case Study
Researched UPI and Online Payment Fraud, wrote a detailed case study covering how the crime works step by step, who is targeted, and the consequences. Created a prevention checklist with tips specific to college students in India.
